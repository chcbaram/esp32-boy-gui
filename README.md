# esp32-boy-gui
 


## 개요 
- ESP32-S3-BOY 펌웨어 업데이트를 위한 GUI 프로그램 

## 개발환경
- python3
- PySide6 6.5.0

## 설치하기
- 패키지 설치
  ```
  pip install -r requirements.txt
  python setup.py install

  or
  
  pip install -r requirements.txt
  python3 setup.py install
  ```
